PicLift

HTML, CSS, JS, Python, Google Search API, Atom, Postman, Django, AWS

Created at SuperPosition IV, an all female hackathon, PicLift provides users with pictures of things the like to help them destress. Me and my team learned how to implement many different programs to create a cohesive, functioning web application. More specifically, we learned how to write a formal plan including the architecture, schema, and sequence diagram. We also learned how to connect front-end and back-end code. Ultimately, this project won Best Web Application at the hackathon.
